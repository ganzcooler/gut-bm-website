# Plan: GuT Mayor Candidate Landing Page

## Context
- **Client**: Mayoral candidate affiliated with "GuT" voter group
- **Colors**: Yellow (`#FFD700` or similar) and Blue (`#0055A4` or similar) — matching the municipality
- **Navigation**: Only "Legal Notice" and "Privacy Policy" links (no other navigation)
- **Tone**: Civic, trustworthy, modern yet approachable

## Aesthetic Direction
Bold editorial/magazine aesthetic with asymmetric layouts. Strong typographic hierarchy. Yellow/blue palette with dominant yellow energy and blue structural anchoring. Clean, confident, memorable.

## Typography
- **Display**: Something bold and distinctive — e.g., "Playfair Display" or "DM Serif Display" for gravitas
- **Body**: Clean geometric sans — e.g., "IBM Plex Sans" or "Manrope"

## Technical Approach
Single-page HTML/CSS/JS implementation. No framework dependency.

## Structure

### Sections (top to bottom)
1. **Hero** — Candidate name, tagline, visual energy, CTA
2. **About** — Brief bio/position statement
3. **Values/Issues** — Key priorities (3-4 items)
4. **Contact/Footer** — Minimal footer with only Legal Notice + Privacy Policy links

### Page Structure
```
[Hero - full viewport, bold typography, geometric accents]
[Voting symbol / GuT branding element]
[About section - asymmetric layout]
[Issues section - card grid]
[Footer - Legal Notice | Privacy Policy]
```

## Implementation Steps

1. Create `index.html` with semantic HTML structure
2. Add inline CSS with CSS variables for yellow/blue palette
3. Implement animations (staggered reveals on load, scroll-triggered fade-ins)
4. Add geometric decorative elements (SVG shapes, lines)
5. Create placeholder content for:
   - Candidate name
   - Tagline
   - Bio
   - 3-4 key issues/positions
6. Footer with exactly two links: "Legal Notice" and "Privacy Policy"
7. Create placeholder `legal-notice.html` and `privacy-policy.html` pages

## Files to Create
- `index.html` — main landing page
- `legal-notice.html` — legal notice page
- `privacy-policy.html` — privacy policy page

## Notes
- Colors will be CSS variables: `--primary-yellow`, `--primary-blue`
- No JavaScript framework, vanilla JS for animations
- Responsive design (mobile-first breakpoints)
- Content is placeholder using "Max Mustermann" as candidate name — user can customize
